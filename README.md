# 日本酒 出荷・コンテナ重量計算ツール

日本酒の瓶・ケースごとの出荷重量と、必要なコンテナ数を自動計算するWebアプリです。
社内の出荷業務効率化のために作成しました。

## 📱 利用方法
以下のURLをクリックすると、ブラウザ上でアプリが起動します（インストール不要）。
**[https://youka419.github.io/sake-weight-calc/](https://youka419.github.io/sake-weight-calc/)**

## 📊 計算の前提データ
本アプリは以下の数値に基づいて計算しています。

| 容量 | 入数/箱 | 1本あたり総重量(瓶込) | 箱のみ重量 |
| --- | --- | --- | --- |
| 1800ml | 6本 | 2.80 kg | 1.70 kg |
| 750ml | 6本 | 1.30 kg | 1.20 kg |
| 720ml | 12本 | 1.14 kg | 0.67 kg |
| 500ml | 12本 | 0.94 kg | 0.00 kg |
| 300ml | 24本 | 0.55 kg | 0.00 kg |
| 180ml | 30本 | 0.34 kg | 0.00 kg |

※数値に変更がある場合は、htmlファイルを修正してください。
